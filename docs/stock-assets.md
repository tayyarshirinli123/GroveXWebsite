# Stock Assets

This document tracks stock imagery used on the GroveX website.

## Current Status

The site currently uses CSS gradients and design-driven layouts rather than stock photography.
When stock images are added, they should be stored in `/public/images/stock/` and documented below.

## Image Guidelines

- Source: Unsplash or Pexels (free for commercial use)
- Theme: Greenhouse tomato cultivation, greenhouse infrastructure, crop monitoring, agronomy inspection
- Avoid: Generic open-field farming, generic smiling office teams, fake laboratory visuals
- Format: WebP or JPEG, optimized for web
- Use next/image for automatic optimization
